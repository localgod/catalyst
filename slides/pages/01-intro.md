# Overview

Catalyst is a JavaScript library designed to facilitate the conversion of C4 diagrams written in [PlantUML](https://plantuml.com/) format into [draw.io](https://draw.io) C4 diagrams. While PlantUML itself is not required as a runtime dependency, the library parses diagrams written in PlantUML's C4 syntax (.puml files). This project uses the ELK (elkjs) layout engine: layered for hierarchical C4 diagrams, force for hub-and-spoke Context diagrams.
